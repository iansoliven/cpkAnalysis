Project Highlights

  • High-throughput STDF ingestion with strict flag filtering.
  • Comprehensive Cp/Cpk statistics with configurable limit precedence.
  • Automatic Excel workbook authoring with histogram, CDF, and time-series charts.
  • Extensible post-processing menu for updating limits and refreshing charts after the main pipeline completes.

Where to Start

If you are new to the solution, read through Getting Started to configure prerequisites and run your first analysis. The post-processing topic explains how to take advantage of the new menu-driven tooling once a workbook is generated.
